# Self-Branding Domain — Shared Variables

## Brand Building Platforms
```
PROFESSIONAL:
  - LinkedIn (primary for B2B, career, consulting, thought leadership)
  - Twitter/X (opinions, real-time commentary, developer/tech community)
  - Substack (newsletter-first audience building)
  - Personal website / portfolio site

VISUAL/CREATIVE:
  - Instagram (lifestyle, design, creative industries, coaching)
  - Dribbble / Behance (designers, developers)
  - GitHub (developers — contribution graph as portfolio)
  - YouTube (long-form authority, tutorials, documentary-style)

COMMUNITY:
  - Reddit (niche expertise, authentic engagement)
  - Discord / Slack communities (deep community plays)
  - Indie Hackers (founders, product builders)
  - Product Hunt (launchers, makers)

SPEAKING / PR:
  - Conference speaker submissions
  - Podcast guest databases
  - HARO (Help a Reporter Out) for media quotes
  - Guest blog posts
```

## Personal Brand Metrics
```
REACH:
  - LinkedIn followers / connections
  - Twitter/X followers
  - Email subscribers
  - YouTube subscribers
  - Podcast listeners

ENGAGEMENT:
  - Average post engagement rate
  - Profile views per week (LinkedIn)
  - DMs / inbound inquiries per month
  - Comment quality (depth of conversation)

AUTHORITY:
  - Inbound speaking invitations
  - Podcast guest invitations
  - Media mentions / quotes
  - Backlinks to personal site
  - "They're the go-to person for X" mentions

OPPORTUNITY:
  - Inbound job offers / consulting inquiries per month
  - Partnership / collaboration proposals
  - Revenue from personal brand (consulting, speaking, courses, products)
  - Career advancement attributed to brand visibility
```

## Brand Positioning Dimensions
```
EXPERTISE AXIS:
  - Generalist (broad authority in a field)
  - Specialist (deep authority in a niche)
  - Intersectionist (unique combination: e.g., "marketing + engineering")

AUDIENCE AXIS:
  - Peer-to-peer (professionals talking to professionals at same level)
  - Practitioner-to-student (expert teaching beginners)
  - Leader-to-industry (thought leader shaping the field)

PERSONALITY AXIS:
  - Educator (teaches frameworks and principles)
  - Storyteller (shares experiences and lessons)
  - Analyst (data and research-driven insights)
  - Provocateur (challenges conventional thinking)
  - Connector (brings people and ideas together)

CONTENT AXIS:
  - How-to (practical, tactical, actionable)
  - Why (strategic, philosophical, big picture)
  - What's next (predictions, trends, futures)
  - Behind-the-scenes (process, reality, authenticity)
```

## LinkedIn Profile Sections
```
PROFILE:
  - Profile photo (professional, warm, high resolution)
  - Banner image (brand statement or visual identity)
  - Headline (150 chars — not just job title)
  - About section (2,600 chars — story + positioning + CTA)
  - Featured section (3-5 best posts, articles, or links)
  - Experience (impact-focused, not job description)
  - Education
  - Skills (top 5 matter, rest are noise)
  - Recommendations (quality > quantity)
  - Creator mode (if publishing content)
```

## Portfolio Types
```
B2B / CONSULTING:
  - Case studies with measurable results
  - Client testimonials
  - Process documentation
  - Methodology / framework documents
  - Speaking clips
  - Published articles / guest posts

CREATIVE / DESIGN:
  - Project galleries
  - Before/after transformations
  - Process walkthroughs
  - Client briefs + outcomes
  - Personal projects (shows taste and initiative)

TECHNICAL:
  - GitHub repositories with READMEs
  - Open source contributions
  - Technical writing (blog posts, docs)
  - Side projects / products built
  - Conference talks / demos

LEADERSHIP / EXECUTIVE:
  - Narrative case studies (not just what you did — the situation, decision, outcome)
  - Team / company outcomes under your leadership
  - Board presentations / white papers (sanitized for public)
  - Board / advisory positions
  - Media interviews
```

## Visibility Channels
```
SPEAKING:
  - Local meetups (starting point)
  - Industry conferences (intermediate)
  - Keynote conferences (advanced)
  - Virtual summits (accessible entry)
  - Internal company talks (often overlooked)
  - Podcasts as guest (fastest path to authority)

MEDIA:
  - Trade publications (niche authority)
  - National publications (broad reach)
  - Industry newsletters (targeted)
  - Podcast interviews
  - YouTube interviews / collaborations

PARTNERSHIPS:
  - Content collabs with adjacent creators
  - Co-authored research / reports
  - Joint webinars
  - Cross-newsletter promotions
  - Affiliate / referral relationships
```
