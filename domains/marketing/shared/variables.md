# Marketing Domain — Shared Variables

## Channels
```
PAID_CHANNELS:
  - Google Ads (Search, Display, Shopping, YouTube)
  - Meta Ads (Facebook, Instagram)
  - LinkedIn Ads (Sponsored Content, InMail, Lead Gen)
  - TikTok Ads
  - Twitter/X Ads
  - Programmatic (DSPs: DV360, The Trade Desk)
  - Spotify / Podcast Ads
  - Connected TV (CTV)
  - Amazon Ads

OWNED_CHANNELS:
  - Website / Blog
  - Email list
  - SMS list
  - Push notifications
  - App (if exists)
  - YouTube channel
  - Podcast

EARNED_CHANNELS:
  - Organic social (LinkedIn, Twitter/X, Instagram, TikTok)
  - SEO / organic search
  - PR / media coverage
  - Word of mouth / referral
  - Reviews (G2, Trustpilot, App Store)
  - Community (Reddit, Slack, Discord)
```

## Core Marketing Metrics
```
AWARENESS:
  - Impressions
  - Reach
  - Share of Voice (SOV)
  - Brand search volume
  - Direct traffic

ACQUISITION:
  - CTR (Click-Through Rate)
  - CPC (Cost Per Click)
  - CPM (Cost Per 1,000 Impressions)
  - CPL (Cost Per Lead)
  - CAC (Customer Acquisition Cost)
  - Conversion Rate (CVR)
  - Landing Page Conversion Rate

ENGAGEMENT:
  - Engagement Rate
  - Time on Site
  - Pages per Session
  - Scroll Depth
  - Video View Rate
  - Email Open Rate
  - Email Click Rate
  - Reply Rate

REVENUE:
  - MQL → SQL Rate
  - SQL → Close Rate
  - Pipeline Generated ($)
  - Revenue Influenced ($)
  - ROAS (Return on Ad Spend)
  - ROI
  - LTV (Lifetime Value)
  - Payback Period

RETENTION:
  - Churn Rate
  - NPS (Net Promoter Score)
  - Repeat Purchase Rate
  - MRR / ARR
```

## Funnel Stages
```
TOFU (Top of Funnel — Awareness):
  - Channels: organic social, paid awareness, PR, SEO
  - KPIs: impressions, reach, brand awareness lift
  - Content: educational, problem-aware, broad interest

MOFU (Middle of Funnel — Consideration):
  - Channels: retargeting, email nurture, search (non-brand), content
  - KPIs: leads, MQLs, demo requests, content downloads
  - Content: comparison, solution-aware, use cases, social proof

BOFU (Bottom of Funnel — Decision):
  - Channels: branded search, sales outreach, retargeting, email
  - KPIs: SQLs, opportunities, closed won, pipeline $
  - Content: case studies, pricing pages, trials, demos
```

## Budget Allocation Frameworks
```
STARTUP (< $50K/mo):
  - 40% performance / direct response
  - 30% content + SEO
  - 20% email + nurture
  - 10% experiments

GROWTH ($50K–$500K/mo):
  - 50% performance (proven channels)
  - 20% brand awareness
  - 20% content + SEO
  - 10% experiments

ENTERPRISE ($500K+/mo):
  - 40% brand
  - 35% performance
  - 15% content + community
  - 10% partnerships + events
```

## Positioning Variables
```
POSITIONING_AXES:
  - Price: premium ↔ budget
  - Audience: enterprise ↔ SMB ↔ consumer
  - Approach: DIY ↔ done-for-you
  - Speed: instant ↔ deep
  - Trust: established ↔ challenger
  - Category: creator ↔ follower ↔ differentiated

COMPETITIVE_MOATS:
  - Network effects
  - Data advantage
  - Switching costs
  - Brand equity
  - Distribution
  - Technology / IP
  - Cost structure
```
