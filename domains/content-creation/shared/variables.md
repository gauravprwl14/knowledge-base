# Content Creation Domain — Shared Variables

## Content Types
```
WRITTEN:
  - Blog posts (500-2500 words)
  - Long-form guides (2500-10,000 words)
  - Whitepapers (5-20 pages)
  - Case studies (500-2000 words)
  - Newsletters
  - LinkedIn articles
  - Twitter/X threads
  - Substack posts
  - Email courses
  - Opinion pieces / essays

VISUAL:
  - Infographics
  - Data visualizations
  - Carousels (LinkedIn, Instagram)
  - Slide decks / presentations
  - Reports (designed PDF)

AUDIO / VIDEO:
  - Podcast episodes (20-60 min)
  - Short-form video (60 sec — TikTok, Reels, Shorts)
  - Long-form video (5-30 min — YouTube)
  - Webinars / live streams
  - Video essays
  - Course modules

INTERACTIVE:
  - Quizzes
  - Calculators
  - Templates (downloadable)
  - Checklists
  - Frameworks (visual PDFs)
```

## Content Metrics
```
REACH:
  - Impressions (views)
  - Unique readers
  - Followers / subscribers gained
  - Share of voice in topic

ENGAGEMENT:
  - Likes, comments, shares, saves
  - Engagement rate (engagements / impressions)
  - Comments per post (quality signal)
  - Saves / bookmarks (intent signal)
  - Email open rate, click rate
  - Time on page
  - Scroll depth

CONVERSION:
  - Newsletter signups from content
  - Leads generated from content
  - Content → trial / purchase attribution
  - Demo requests from blog/SEO

AUTHORITY:
  - Backlinks earned
  - Media mentions
  - Podcast invitations received
  - Speaking invitations
  - Follower growth rate
```

## Content Formats by Platform
```
LINKEDIN:
  - Optimal post length: 150-300 words (no link in post)
  - Best formats: text-only, carousel (PDF), video, poll
  - Posting cadence: 3-5x/week for growth
  - Best time: Tue-Thu, 7-9am or 12-1pm local
  - Hook requirement: first 2 lines must earn "see more" click
  - Character limit: 3,000 characters

TWITTER/X:
  - Single tweet: 280 characters
  - Thread: 5-15 tweets for maximum engagement
  - Best formats: takes/opinions, threads, quote tweets
  - Posting cadence: 1-5x/day
  - Best time: 8-10am, 12pm, 5-6pm

INSTAGRAM:
  - Caption: 125 chars before "more" cutoff — lead with hook
  - Optimal hashtags: 3-5 relevant tags
  - Best formats: Reels (entertainment + education), Carousels (save-worthy lists)
  - Stories: 24hr ephemeral, behind-the-scenes

YOUTUBE:
  - Title: 50-70 chars — keyword + hook
  - Thumbnail: high contrast, clear face/text, promise implied
  - Description: keyword-rich first 150 chars (above fold)
  - Best length: 8-15 min for education (higher ad revenue, retention)
  - Posting cadence: 1-2x/week for algorithm growth

TIKTOK:
  - Optimal length: 15-60 sec for discovery; up to 3 min for established creators
  - Hook: first 1-3 seconds determine retention
  - Captions: add for accessibility + silent viewing
```

## Content Production Stages
```
IDEATION → BRIEF → RESEARCH → OUTLINE → DRAFT → EDIT → OPTIMIZE → PUBLISH → DISTRIBUTE → MEASURE
```

## Reading Level Targets
```
General consumer: Grade 6-8
Professional B2B: Grade 10-12
Technical / developer: Grade 12-14 (specificity over simplicity)
Executive: Grade 8-10 (concise, data-heavy, no fluff)
```
